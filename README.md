# TagSale
Steps to run the code in this repository:-

i) Download xampp server package for hosting the MySQL and php server in localhost from the following link-
https://www.apachefriends.org/download.html

ii) Install it in your system. Make sure the location is C://xampp/..

iii) Download the zip file of code from this reposirtory, extract it and move the TagSale directory to C://xampp/htdocs/..
The final location shall be like- C://xampp/htdocs/TagSale/..

iv) Open the xampp-control.exe app!

v) Start the Apache and SQL server!

vi) Now, visit http://localhost/TagSale/
